# Референс-плейбук: на кого опирается Marketer

Живой документ. Не разовая справка — пополняется каждый раз, когда Data Collector или Marketer находят проверенный паттерн (реальные цифры, а не мнение). Источник появления: владелец прямо потребовал, чтобы Marketer учился на конкретных примерах, а не изобретал хуки с нуля (инцидент 2026-07-10 — несколько подряд отклонённых углов подачи).

## Принцип отбора источников

Три типа, никаких непроверенных "топ-10 маркетологов" из SEO-подборок:

1. **Эксперты с проверяемой специализацией** — публичная методология, кейсы с цифрами.
2. **Бренды-аналоги** — реальные конкуренты/близкие по позиционированию, которых можно системно наблюдать.
3. **Проверяемая аналитика** — отчёты с данными, не мнения.

## 1. Эксперты

### Анна Ковтун — маркетплейсы

- Специализация: Ozon/WB, автор курса «Как стать продавцом на Ozon».
- Принцип: «Контент — отражение позиционирования и потребностей, которые продукт закрывает для аудитории». Контент не работает в отрыве от товара/упаковки/оффера — только в связке.
- Кейс: категория детской одежды — доработка товара + упаковки + контента + оффер с подарком + допродажи → рост продаж почти вдвое (3,7 → 6,7 млн ₽/мес).
- Применение к QWEAR: бриф Marketer должен явно указывать, какую потребность ЦА закрывает контент, а не только "красивый хук".

### Саша Гончарова — соцсети как бизнес-инструмент

- Фаундер агентства Sence.Prod и школы NewSence.
- Принцип: не копировать чужую подачу, вести аккаунт органично бренду. Контент оценивается по результату (охваты/выручка/вовлечённость), не по "красиво получилось".
- Применение к QWEAR: SMM weekly report должен явно смотреть на результат (сохранения/переходы/заказы), не на "нравится ли нам эстетика".

## 2. Бренды-аналоги (мониторить через Data Collector)

- **ONSI** — мягкий женский трикотаж, позиционирование "база, которую реально носят" (не пылится в шкафу), спокойная палитра (матча, молоко, пыльно-голубой). Прямой аналог философии QWEAR "идеальная база для каждого дня".
- **Clementina** — кейс VK Ads: CTR 2.12%, покупки ×3.2, выручка 72 690 → 237 454 ₽. Референс по перформансу в VK (не органика, но полезно как ориентир, если дойдём до платного продвижения).
- Sela, Zolla, Ostin — крупные масс-маркет конкуренты, не прямые аналоги по позиционированию (не quiet luxury), но полезны как ориентир по масштабу и охватам.

## 3. Проверяемая аналитика

- **Студия Чижова, «Бренды в Telegram 2026»** (chizhov.io/telegram2026) — 4 года аналитики, 76 брендов, 18 отраслей. Использовать как ориентир по форматам/частоте, актуальным именно для Telegram.

## Найденные рабочие паттерны (пополняется)

Перенесено из реальных находок Data Collector 2026-07-10 (Apify/TikTok):

- Хук с конкретным числом образов из одной вещи ("N вещей = M образов") — кратно больше охват, чем хук про удобство. Пример: «10 вещей = 42 образа» — 5.6M просмотров.
- Формат "вещь, которая есть почти у каждой, но выглядит по-разному" + вопрос на вовлечение (голосование/комментарии) — выше комментарии и охват, чем обзор/отзыв. НО: варианты для голосования должны реально отличаться по стилизации (аксессуары, слои), не только фоном/локацией — иначе сравнение бессмысленно (см. инцидент "кафе vs улица").
- Прямой product-hunt формат ("артикул по шапке профиля") даёт стабильные 300-500k без глубокой подачи — не наш стиль (не quiet luxury), но полезно знать как контрпример.

## Правило применения

Marketer не пишет бриф "из головы". Перед брифом — проверка: есть ли релевантный паттерн в этом документе или у Data Collector. Если нет — Data Collector дособирает (см. `agents/marketer-strategist.md`, раздел "Проверка Работы Data Collector").
