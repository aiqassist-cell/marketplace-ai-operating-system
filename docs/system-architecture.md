# Архитектура Системы

## Управляющий слой

- Менеджер / Project Lead
- Маркетолог-стратег
- SMM / Distribution / World Manager
- Финансист
- Performance-аналитик

## Продуктово-маркетплейсный слой

- Продуктолог
- Логист
- Специалист по фото и фотоворонке
- Digital Stylist / Product Solution Designer

## Контент-завод / Production

- Creative Lead / сценарист
- AI Production Lead
- AI Creator
- Editor / Motion / Post-production
- Account Network / технический специалист

## Поток работы

```text
Продуктолог -> товарные инсайты, боли, отзывы, конкуренты
Маркетолог -> ЦА, позиционирование, оффер, бизнес-гипотеза
Digital Stylist -> готовые решения, правила, образы, комплекты
SMM -> тренды, хуки, форматы, публикации, дистрибуция, обратная связь
AI Production -> генерация, сцены, промпты, адаптации
Фото/фотоворонка -> карточка, визуалы, A/B-гипотезы
Финансист -> маржа, бюджет, экономика масштабирования
Менеджер -> процесс, сроки, статусы, контроль
```

## Первый Контур

Агенты первого контура закрывают критичные функции, где нет или не хватает штатной единицы:

- Productologist - товарная логика и решения по ассортименту.
- Marketer Strategist - коммерческий смысл, ЦА, оффер и воронка.
- SMM / Distribution / World Manager - соцсети, тренды, хуки, публикации, winners и обратная связь аудитории.

SMM не является второстепенным production-агентом. Для QWEAR он стоит рядом с маркетологом, потому что именно он превращает смыслы и rich-контент в регулярный внешний трафик и рыночные сигналы.
