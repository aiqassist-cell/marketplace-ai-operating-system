# Pilot: Brown Basic Dress Content Funnel

Дата старта: 2026-06-02

## Рабочее Допущение

Пилот строится на основе текущего rich-контента QWEAR: коричневое базовое платье / платье-футболка / миди-макси с возможностью стилизации в десятки образов.

Данные, которых не хватает:

- артикул WB/Ozon;
- ссылка на карточку;
- цена;
- текущая конверсия карточки;
- добавления в избранное / корзину;
- заказы;
- выкуп;
- остатки;
- размерный ряд;
- состав ткани;
- фактические отзывы / вопросы покупателей.

Пока этих данных нет, агентская цепочка работает как тест упаковки смыслов, а не как финальное коммерческое решение.

## Цель Пилота

Проверить, может ли одна модель быть упакована как рабочая база гардероба через:

- rich-контент;
- соцсети;
- TikTok/Reels/Shorts;
- карточку маркетплейса;
- фотоворонку;
- контентную воронку.

## Главная Гипотеза

Если показать платье не как отдельный товар, а как базу для нескольких сценариев жизни, то вырастут:

- сохранения;
- переходы в карточку;
- добавления в избранное;
- добавления в корзину;
- качество вопросов покупателей;
- потенциально конверсия карточки.

## Позиционирование Модели

```text
Базовое женственное платье для города, поездок и повседневного ритма.
Не требует сложной стилизации, но легко меняет настроение через детали:
ремень, жакет, обувь, слой, сумка, аксессуары.
```

## Productologist Output

### Факт

- На текущих rich-примерах модель показывается как универсальная база.
- Основной акцент: одна вещь дает много образов.
- Уже есть визуальные сценарии: день/вечер, ремень, жакет, многослойность, город.

### Гипотеза

Модель может работать как товар с длинным окном продаж, если подтвердятся:

- комфортная посадка;
- качественная ткань;
- достаточный выкуп;
- низкий риск возвратов по ожидание/реальность;
- понятная размерная сетка;
- возможность расширения в plus size.

### Риски

- Без данных по ткани и посадке нельзя обещать комфорт и регулярную носку.
- Без отзывов нельзя понять реальные возражения.
- Если платье выглядит слишком базово, нужен rich-контент, который доказывает вариативность.
- Если посадка слабая, стилизация не спасет выкуп.

### Что Проверить

- состав и плотность ткани;
- размерный ряд;
- отзывы по посадке;
- возвраты / выкуп;
- фото на разных ростах и размерах;
- наличие товара перед запуском внешнего трафика.

## Marketer Output

### ЦА

Женщины, которым нужна удобная, современная и женственная вещь на каждый день без ощущения "слишком нарядно" или "слишком просто".

### Главное Обещание

```text
Одно платье, которое легко становится основой для разных образов.
```

### УТП

- легко стилизовать;
- комфортно носить;
- подходит для города, поездки, встречи, офиса и повседневного ритма;
- меняет настроение через детали;
- работает не на один выход, а на повторное использование.

### Оффер Для Теста

```text
Сохраните в гардероб базу, которую можно носить снова и снова.
```

### Главное Возражение

```text
Не будет ли платье слишком простым?
```

### Как Закрываем

Показываем 5-7 образов и простые действия:

- сменить обувь;
- добавить ремень;
- добавить жакет;
- добавить джемпер / свитшот;
- собрать городской образ через кеды, шоппер и очки.

## Digital Stylist Output

### Rich-Контент

1. Одно платье - десятки образов.
2. Платье, которое не требует сложных решений.
3. Меняйте детали - меняйте настроение.
4. От дня к вечеру.
5. Хотите другой силуэт? Добавьте ремень.
6. Одно движение - образ уже другой: добавьте жакет.
7. Для города и на каждый день: кеды + шоппер + очки.

### Сценарии Носки

- город на каждый день;
- прогулка;
- встреча / кафе;
- smart casual;
- поездка;
- прохладная погода;
- день/вечер.

## SMM Output

### Площадки Для Первого Теста

- TikTok или Reels как основная короткая площадка;
- Telegram/VK как место для объяснения и ссылки на карточку.

### Хуки

1. Одно платье - 5 образов за 15 секунд.
2. Как сделать базовое платье не скучным.
3. Добавила ремень - и силуэт стал другим.
4. День против вечер: меняем только детали.
5. Ошибка, из-за которой базовое платье выглядит домашним.
6. Платье, которое работает без сложной стилизации.
7. Что надеть, когда не хочется собирать сложный образ.

### Первый Контент-Пакет

```text
5 коротких роликов:
1. 5 образов за 15 секунд
2. день / вечер
3. без ремня / с ремнем
4. добавьте жакет
5. городской образ: кеды + шоппер + очки

2 объясняющих поста:
1. почему это база гардероба
2. как выбрать образ под сценарий
```

## AI Production Output

### Production Formats

- оживление rich-слайда;
- вертикальный ролик 9:16;
- быстрые смены образов;
- before/after по силуэту;
- карусель -> motion;
- титры крупно, читаемо на мобильном.

### QA

Проверить:

- цвет платья не меняется;
- длина и посадка не искажаются;
- модель выглядит естественно;
- аксессуары не перекрывают товар;
- текст читается;
- первый кадр понятен без звука.

## Photo Funnel Output

### Карточка / Фотоворонка

1. Главное фото: платье, силуэт, длина, цвет.
2. Главное обещание: одно платье - десятки образов.
3. Галерея вариантов: 5 образов.
4. День/вечер.
5. Без ремня / с ремнем.
6. Жакет / многослойность.
7. Ткань / посадка / размер.
8. Кому подходит.
9. Что купить вместе.
10. CTA: сохранить базу в гардероб.

## Критерии Успеха Первого Теста

Минимальный сигнал:

- есть сохранения;
- есть вопросы по размеру / ткани / где купить;
- есть переходы в карточку;
- один из форматов явно лучше остальных.

Сильный сигнал:

- winner по сохранениям или досмотрам;
- рост избранного / корзины;
- комментарии дают новые возражения;
- rich-паттерн можно переносить на другие модели.
