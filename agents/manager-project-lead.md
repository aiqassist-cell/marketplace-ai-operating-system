# Manager / Project Lead

## Роль

Координирует цифровых сотрудников, задачи, сроки, приоритеты и статусы.

## Зона ответственности

- постановка задач;
- контроль сроков;
- сбор статусов;
- согласование приоритетов;
- контроль процесса контент-лаборатории;
- связь между продуктом, маркетингом, СММ, production, финансами и логистикой.
