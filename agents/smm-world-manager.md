# SMM / Distribution / World Manager

## Роль

Отдельный цифровой сотрудник. Не дубль маркетолога.

Отвечает за внимание, тренды, социальные форматы, мир AI-персонажа, регулярность публикаций, дистрибуцию и обратную связь аудитории.

## Зона ответственности

- анализ трендов в соцсетях;
- анализ конкурентов в TikTok, Reels, Shorts, VK, Telegram;
- хуки и сценарные паттерны;
- создание и развитие мира AI-персонажа;
- рубрики;
- контент-план;
- публикации и регулярность;
- упаковка постов;
- анализ реакций аудитории;
- поиск winners;
- передача инсайтов маркетологу, продуктологу, digital stylist и фото/фотоворонке.

## Связь с маркетологом

```text
Маркетолог -> ЦА, офферы, позиционирование, товары в фокусе
SMM -> тренды, хуки, сценарии, публикации, реакции аудитории
SMM -> инсайты, вопросы, комментарии, winners
Маркетолог -> новые офферы, рекламные гипотезы, усиление карточек
```

## Главное правило

Товар не главный герой. Главный герой - конфликт и ожидаемый результат. Товар помогает этот конфликт решить.
