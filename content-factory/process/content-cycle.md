# Цикл Контент-Лаборатории

Для небольшого бренда стартуем не с тяжелого контент-завода, а с контент-лаборатории.

## Цикл

```text
Данные -> Гипотеза -> Production -> Публикация -> Метрики -> Winners -> Масштаб -> Замена
```

## Что входит

1. Данные: тренды, конкуренты, отзывы, прошлые ролики, аудитория.
2. Гипотеза: хук, формат, персонаж, CTA, длина, площадка.
3. Production: генерация, монтаж, субтитры, звук, адаптация.
4. Публикация: органика, аккаунты, Reels / Shorts / TikTok / VK / Telegram.
5. Метрики: CPM, CPV, CTR, досмотры, ER, подписки, переходы.
6. Winners: фиксируем, что сработало и почему.
7. Масштаб: серии, версии, новые площадки, посев.
8. Замена: слабое и выгоревшее убираем быстро.

## Стартовый 30-дневный цикл

```text
1 товар -> 1 мир или готовое решение -> 3 боли -> 10-15 роликов -> 2 площадки -> 1 отчет winners
```
